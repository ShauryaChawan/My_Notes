## My Code
```java
import java.io.*;
import java.util.*;

public class Solution {

  public static void main(String[] args) {
    Scanner sc = new Scanner(System.in);
    
    int num = sc.nextInt();
    int num2 = sc.nextInt();
    int num3 = sc.nextInt();
    sc.close();
    System.out.println(num);
    System.out.println(num2);
    System.out.println(num3);
  }
}
```

## Optimized Code

```java
import java.io.*;
import java.util.*;

public class Solution {
  public static void main(String[] args) {
    Scanner sc = new Scanner(System.in);
    
    while(sc.hasNext()){
      int num = sc.nextInt();
      System.out.println(num);
    }
    sc.close();
  }
}

```
