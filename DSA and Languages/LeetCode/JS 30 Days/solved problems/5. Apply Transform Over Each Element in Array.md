# 2665. Counter II

```js
var map = function(arr, fn) {
	let arr2 = [];
	for (let i in arr) {
		arr2.push(fn(arr[i]*1, i*1));
	}
	return arr2;
};
```

Optimized Code:

```js
var map = function(arr, fn) {
	return arr.map(fn)
};
```
