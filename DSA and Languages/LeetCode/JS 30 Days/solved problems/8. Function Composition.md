# 2629. Function Composition

```js
var compose = function (functions) {
  return function (x) {
    result = x;
    for (let i = functions.length - 1; i >= 0; i--) {
      result = functions[i](result);
    }
    return result;
  };
};
```

Optimised Code:

```js
var compose = function(functions) {
  if(functions.length===0){
  return function(x) { return x; }

  return function(x){
    let res=x;
```
